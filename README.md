## 4. Project Workflow

### Step 1: Data Loading
- Imported dataset using Pandas
- Inspected dataset structure and summary

### Step 2: Exploratory Data Analysis (EDA)
- Summary statistics
- Distribution analysis
- Correlation analysis
- Trend identification
- Data visualization

### Step 3: Data Cleaning
- Treated missing values
- Removed duplicate records
- Corrected inconsistent formatting
- Converted data types
- Created calculated columns

### Step 4: SQL Analysis
- Loaded cleaned data into PostgreSQL / MySQL / SQL Server
- Executed queries using:
  - SELECT
  - WHERE
  - GROUP BY
  - ORDER BY
  - JOIN
  - Aggregate functions (SUM, COUNT, AVG)
  - Subqueries

### Step 5: Power BI Dashboard
- Connected Power BI to SQL database
- Built data model
- Designed interactive visual dashboard
- Added KPIs, slicers, and filters

### Step 6: Reporting
- Documented key findings
- Provided insights and recommendations
